Tipos de Cabeceras de la Petición

REQUEST HEADERS

Referer,contiene la dirección de la página web anterior de la que provenía el enlace a la página actual que se siguió. Permite a los servidores identificar de dónde los visitan las personas y utilizar estos datos para realizar análisis.

User-Agent, contiene una cadena característica que permite identificar el protocolo de red que ayuda a descubrir el tipo de aplicación, sistema operativo, provedor del software.
