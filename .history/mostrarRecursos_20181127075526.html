RECURSOS WEB

Para poder utilizar esta herramienta de desarrollo solo deberemos permanecer en una página web a nuestra elección y abrir las Herramientas de desarrollo de Google mediante el atajo del teclado F12

  • Primero entramos en la página web donde queremos ver los recursos
  • A continuación al abrir las herramientas de Desarollo de Google, nos vamos a la pestaña Network
  • Ahora veremos un video como se descargan los recursos del funcionamiento de la web, en este caso del diario AS
  • Como hemos visto en el video anterior, al recargar la página esta hace una petición al servidor para descargar todos los recursos necesarios para mostrar al cliente todas las funcionalidades de dicha página. Además esta herramienta Network nos dice debajo de la consola cuanto tiempo ha tardado en descargar dichos recursos.
  • Este botón nos permite limpiar el registro de recursos descargados.
  • Este botón permite detener la descargar de recursos, ya que la página está continuamente descargando recursos para la página web. También realiza la función inversa e inicia la descarga de recursos.
  • Esta herramienta dispone de una barra de filtros para poder separar y diferenciar los tipos de recursos descargados
