TIPOS DE RECURSOS

Los recursos descargados por la web para su correcto funcionamiento son clasificados en tipos. Pero al ser muy numeroso la cantidad de recursos y el gran listado de ellos,
nos facilita tener estos clasificados y ordenados, por ello a la hora de su búsqueda, a través de las herramientas de desarrollo de Google Chrome podemos filtrar los recursos según su tipo.

Para ello deberemos hacer lo siguiente:

  1. Abrimos una página web cualquiera (en mi caso es el diario AS) y lanzamos las herramientas de Chrome mediante el atajo F12 de nuestro teclado.
  2. A continuación al abrir las herramientas de Desarollo de Google, nos vamos a la pestaña Network
  3. Ahora veremos un video como se descargan los recursos del funcionamiento de la web, en este caso del diario AS.
  4. Como hemos visto en el video anterior, al recargar la página esta hace una petición al servidor para descargar todos los recursos necesarios para mostrar al cliente todas las funcionalidades de dicha página. Además esta herramienta Network nos dice debajo de la consola cuanto tiempo ha tardado en descargar dichos recursos.
  5. Pues bien, ahora para poder clasificar todos los archivos descargados,podremos filtrar estos accediendo a su tipo al que pertenecen, como vemos aquí. Por defecto viene marcada la casilla All que nos muestra todos los recursos descargados.
  6. Si por ejemplo queremos ver los archivos JavaScript descargados marcamos la casilla JS como hacemos aquí.

BOTONES PESTAÑA NETWORK

    DESCRIPCIÓN DE TIPOS

    XHR Es una interfaz empleada para realizar peticiones HTTP y HTTPS a servidores Web
    JS Es un lenguaje de programación utilizado para el lado del cliente, permitiendo crear efectos atractivos y dinámicos a las páginas web
    CSS Son archivos de hojas de estilo aplicadas a un documento estructurado como puede ser HTML
    Img Son las imágenes de la web definidas en el archivo html
    Media Corresponde al contenido multimedia del que dispone la web ya sea audio o video
    Font Son archivos de la fuente utilizada en la web con su tamaño y estilo
    Doc Documentación de los archivos html de la página web
    WS Contiene información sobre el dominio de la página web
    Manifest Son archivos JSON, que permite que podamos controlar cómo se muestra la página al usuario en áreas donde normalmente ven páginas nativas
    Other Aquí se muestran otros recursos como pueden ser GIFs o archivos JSON
