INSPECCIONADOR DE ELEMENTOS

Podemos iniciar esta herramienta mediante dos métodos, el primero sería abriendo las herramientas de desarrolladores ya en nuestro navegador pulsando F12 en nuestro teclado o haciendo click derecho en la página y pulsando inspeccionar.

    Ver el código fuente de la página y modificarlo

    1. Entramos en la pestaña 'Elements'.
    2. Se nos abrirá la consola con el html perteneciente a la página que tengamos abierta en el navegador como vemos aquí.
    3. Una vez aquí el 'h2' perteneciente al título de esta noticia podremos modificarlo haciendo click derecho sobre el título y hacemos click en Edit text donde cambiaremos el texto y quedará así.
    4. También podemos añadir un elemento o un segundo título a la noticia, de la misma forma que el paso anterior pero clickando sobre 'Edit as HTML' en este ejemplo añado un 'h3' con el texto : Y Vinicius pa'cuando.

    Ver el css asociado a un elemento de la página, indicar las reglas css que afectan a dicho elemento y modificar el estilo css online

    1. Iniciamos las herramientas de desarrollo con el acceso rápido de "F12".
    2. Hacemos click sobre la pestaña de "Sources".
    3. Accedemos a la carpeta css de la página y seguidamente al archivo css transforms2D.css en el cuál a su derecha podemos ver el contenido css de la página en cuestión.
    4. En este caso podemos ver que se tratan de cuadrados, y cogemos como ejemplo el cuadrado 1. Para asignar la dimensión de los cuadrados los seleccionamos mediante un selector de clases con el .cuadrado. Los cuáles se les aplica 200 px de ancho y 200px de alto. En el caso de la imágen corresponde a la zona rodeada de azul. Y para aplicar un estilo al cuadrado, lo hacemos mediante el selector ID #Cuadrado1. Este ID viene dado anteriormente en el documento html. Podemos ver el ejemplo en la imágen anterior (redondeado de color rojo) como se le aplica al cuadrado un color verde de fondo.
    5. Para poder modificar el css desde la consola, solo debemos hacer doble click sobre el elemento que queremos modificar y seleccionar el efecto que queremos aplicarle. En este caso cambio el color de fondo del cuadrado1 a el color morado.
